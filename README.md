# AI Health Insurance ML Model

## Overview
This project focuses on developing a Machine Learning Model for Health Insurance. The aim is to leverage AI to predict health insurance claims and streamline the decision-making process for insurance providers.

## Features
- **Predictive Analytics**: Utilizes machine learning algorithms to predict healthcare costs.
- **Data Visualization**: Tools for visualizing data trends and insights.
- **User-Friendly Interface**: Simplified access to model predictions.

## File Structure
```
AI-Health-Insurance-MLModel/

│   └── model_training.ipynb

└── README.md
```

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/akhilerla9/AI-Health-Insurance-MLModel_V1.24-03-2026.git
   cd AI-Health-Insurance-MLModel_V1.24-03-2026
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Load the data in `notebooks/data_preprocessing.ipynb` for preprocessing.
2. Train the model using `notebooks/model_training.ipynb`.
3. Use `src/model.py` for predictions.

## Model Details
- **Algorithms Used**: Random Forest, Gradient Boosting, etc.
- **Performance Metrics**: Accuracy, Precision, Recall, ROC-AUC.
